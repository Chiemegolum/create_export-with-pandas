# 🐼Create and Export in Pandas

## 🔍Project Overview
 A detailed creation of a dataframe and its exportaion as a CSV file in Pandas using markdowns.

## ⏳Project Task
For this exercise, you will create a dataframe and export it as a CSV file. The data frame will contain the details of imaginary students in a high school.
- Name of student
- Surname of Student
- Date of birth of student
- Students Career option ( Art of Science)
- Age of students ( to test column operation)

Requirements
- The Dataframe must contain 5 columns and 20 rows.
- The Dataframe must contain an index.
- Students must export dataframe as a CSV file.

## 🛠Tools Used
- Python (Pandas), Jupyter Notebook
